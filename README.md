# Dizcovery

##About

This application allows a user to view viral content based on their interests, analyzes consumer trends and visualizes data.

##Technologies Used 
* Mongo DB 
* Express
* Angular
* Node.js
* D3.js
* Google Big Query API 
* Reddit API 
* Embedly API
